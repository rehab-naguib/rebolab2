# Foobar
**Foobar is a python library for dealing with word pluraliztion**
## Installtion
Use the package manager [pip](https://pip.com) to intall footbar
```
 pip install footbar
```
## Usage
```diff
- import foobar

footbar.pluralize('word') 
# returns 'word'
footbar.pluralize('goose')
# returns 'goose'
footbar.singularize('phenomena')
# returns 'phenomena'
```
